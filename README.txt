路亚鱼种百科网站 - 使用说明

这是一个关于路亚鱼种介绍的静态网站，包含前台展示和后台管理功能。

网站文件结构：
- index.html       # 前台首页，展示鱼种信息
- admin.html       # 后台管理页面，管理鱼种数据
- css/style.css    # 网站样式文件
- js/app.js        # 前台JavaScript逻辑
- js/admin.js      # 后台管理JavaScript逻辑
- data/fish-data.json # 鱼种数据文件（默认包含2条鱼的数据）
- images/          # 图片目录（当前为空，可添加鱼种图片）

如何使用：
1. 双击 index.html 在浏览器中打开前台网站
2. 点击导航栏中的"管理后台"或直接双击 admin.html 打开管理后台

注意事项：
1. 由于浏览器安全限制，如果直接从文件系统打开（file://协议），某些功能可能受限。
   建议使用现代浏览器（Chrome、Edge、Firefox），并在打开时允许加载本地资源。

2. 如果网站无法加载数据，请检查浏览器的控制台是否有错误信息。
   可以尝试以下方法：
   - 使用浏览器打开时，添加 --allow-file-access-from-files 参数（仅Chrome）
   - 或将网站文件夹放到本地服务器中运行

3. 管理后台功能：
   - 添加新鱼种：点击"添加新鱼种"按钮
   - 编辑鱼种：在表格中点击"编辑"按钮
   - 删除鱼种：在表格中点击"删除"按钮
   - 导出数据：将当前数据导出为JSON文件
   - 导入数据：从JSON文件导入数据

4. 数据保存：
   - 所有修改的数据会自动保存到浏览器的本地存储（localStorage）
   - 建议定期使用"导出数据"功能备份数据
   - 导出的数据可以在其他设备上使用"导入数据"功能恢复

默认鱼种数据：
网站已包含两条鱼的示例数据：
1. 鲈鱼 - 常见路亚目标鱼，中等难度
2. 鳜鱼 - 名贵淡水鱼种，高难度

如需添加图片：
1. 将鱼种图片放入 images/ 文件夹
2. 在管理后台编辑鱼种时，在"图片URL"字段填写图片路径，如：images/bass.jpg

网站功能：
- 响应式设计，支持手机和电脑访问
- 鱼种详细信息查看
- 完整的后台管理功能
- 数据导入导出
- 本地数据存储

技术支持：
这是一个纯前端静态网站，无需服务器或数据库。
所有代码都是开源且可自定义修改的。
